# Sales Analysis Dashboard (Excel)

## 📊 Overview

This project focuses on analyzing sales data using Excel to understand business performance, trends, and category contribution.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Slicers

## 📌 Key Features

* Region-wise sales analysis
* Category-wise performance tracking
* Interactive dashboard using slicers

## 📈 Insights

* Identified high-performing regions
* Detected low-profit categories
* Analyzed sales trends for decision-making

## 🎯 Conclusion

This dashboard helps in understanding sales performance and supports data-driven business decisions.
